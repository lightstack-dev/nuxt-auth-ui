# Authentication UI for Nuxt

Beautiful, production-ready components powered by [Logto](https://logto.io/).

## Quick Start

```bash
npm install @lightstack/nuxt-auth-ui @logto/nuxt
```

```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  modules: [
    '@logto/nuxt',
    '@lightstack/nuxt-auth-ui'
  ]
})
```

```vue
<template>
  <ASignInButton />
</template>
```
